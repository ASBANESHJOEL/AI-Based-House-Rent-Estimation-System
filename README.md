# AI-Based House Rent Estimation System

A Machine Learning web application that predicts monthly house rent based on:

- Locality
- BHK
- Square Feet
- Furnishing Status

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- Streamlit
- Random Forest Regressor
